# 传输层

## 前言

传输层定义了主机应用程序之间端到端的连通性。传输层中最为常见的两个协议费根鞋是传输控制协议TCP和用户数据包协议UDP

### 传输层概述

* 传输层提供端到端的连接
* 网络层提供点到点的连接

### 传输层协议

| 协议 |                             备注                             |
| :--: | :----------------------------------------------------------: |
| TCP  | 传输控制层协议：可靠的面向连接的协议。传输效率低，类似打电话 |
| UDP  | 用户数据报协议：不可靠的，无连接的服务。传输笑了高，类似群聊 |
