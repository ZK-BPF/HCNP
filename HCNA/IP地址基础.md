# IP基础

* 在IP网络中，通讯节点需要有IP地址
* 以点分十进制标识，由32位二进制组成
* 分为网络位和主机位两部分
  * 网络位，代表IP地址所属的网段
  * 主机位，代表网段上的某一节点
  * 由子网掩码决定分界点

## 子网掩码

* 用于区分网络部分和主机部分
* 1表示网络位，0表示主机位

### 网络地址、主机地址、广播地址

* 每个网段上都有两个特殊地址（最小和最大）不能分配给节点

![1595650110106](IP地址基础.assets/1595650110106.png)

| 网络地址 | 主机位全为0代表一个网段（头）                  |
| -------- | ---------------------------------------------- |
| 主机地址 | 主机位不全为0也不全为1代表一个网段内的一个节点 |
| 广播地址 | 主机位全为1代表一个网段内的所有节点（尾）      |
